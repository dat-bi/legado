# Web Legado
 Sử dụng Vue3 cho kệ sách web và trình chỉnh sửa nguồn web
## Đường dẫn
* http://localhost:8080/ Kệ sách
* http://localhost:8080/#/bookSource Chỉnh sửa nguồn sách
* http://localhost:8080/#/rssSource Chỉnh sửa nguồn RSS
